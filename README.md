# 🧠 MindTrace – AI-Powered Student Mental Fatigue Detector

> An intelligent wellness analytics platform that helps students identify mental fatigue, monitor well-being, and receive personalized recovery recommendations using Machine Learning.

---

## 📖 Overview

Mental fatigue has become a major challenge among students due to increasing academic workloads, prolonged screen exposure, stress, poor sleep habits, and unhealthy lifestyles.

**MindTrace** combines Machine Learning, wellness analytics, and interactive visualizations to evaluate student wellness patterns and predict fatigue levels before they significantly impact academic performance and mental well-being.

The platform transforms daily lifestyle data into actionable insights through:

✅ Fatigue Prediction

✅ Wellness Scoring

✅ Interactive Analytics Dashboard

✅ AI-Powered Recommendations

---

## 🎯 Project Objectives

- Detect student mental fatigue using Machine Learning
- Assess wellness across multiple lifestyle dimensions
- Calculate fatigue and wellness scores
- Visualize wellness trends using interactive charts
- Identify major contributors to fatigue
- Generate AI-powered personalized recommendations
- Encourage healthier study habits and recovery routines

## 🤖 Machine Learning Model

### 🌳 Random Forest Classifier

MindTrace uses a Random Forest Classifier trained on student wellness data to predict fatigue levels.

### Why Random Forest?

✅ High prediction accuracy

✅ Handles multiple wellness indicators

✅ Resistant to overfitting

✅ Supports complex behavioral relationships

✅ Strong classification performance
