# 🚀 MindTrace – AI-Powered Student Mental Fatigue Detector 🧠✨

## 📢 Project Showcase

MindTrace uses Machine Learning, Wellness Analytics, 📊 Radar Charts, 📈 Fatigue Factor Analysis, and 🤖 AI-Powered Recommendations to help students identify mental fatigue early and improve their well-being.

## 🌐 Live Demo

🔗 **https://mind-trace-zb6b.vercel.app/**

## 💡 Features

- 🤖 AI-Powered Fatigue Prediction
- 📊 Interactive Wellness Radar Chart
- 📈 Fatigue Factor Analysis Dashboard
- 📋 Progress Tracker
- 💚 Wellness Score Analysis
- 🎯 Personalized Recommendations
- 📱 Responsive User Interface

## 🛠️ Technology Stack

- Python
- Scikit-Learn
- Pandas
- NumPy
- HTML5
- CSS3
- JavaScript
- Chart.js

## 📚 About

Explore the dashboard, assess wellness patterns, and discover personalized recommendations for healthier study habits, improved productivity, and better student well-being.
